fruit-basket.csv - full verified data
b.csv - bad dataup - results in an error
b2.csv - bad data mockup - results in an error

INSTALL THE APP:
sudo chown -R $USER /usr/local/lib/node_modules or sudo chown -R $USER /usr/local/lib
npm install -g  fruit-basket

RUN IN COMMAND TERMINAL:
- Execute the following command 'fruit-basket' to start the app
- When prompted 'Enter CSV File:', insert 'fruit-basket.csv' in field and hit Enter
- App should execute with proper outputs
